# 🏠 Home Service Marketplace

A Django-based web application for booking and managing household services.  
This project provides a clean, professional dashboard for both users and admins, with workflows for featured services, bookings, and future chat/payment integrations.

---

## ✨ Features
- **Service Listings**: Add, edit, and manage services (e.g., salon, WiFi installation, cleaning).
- **Featured Services**: Highlight selected services on the homepage.
- **Booking Workflow**: Users can book services with confirmation pages.
- **Admin Dashboard**: Role-aware dashboards with filters and management tools.
- **Bootstrap Integration**: Responsive, polished UI for a professional look.
- **Media Support**: Upload and manage service images.
- **Future Plans**:
  - Demo/test payment flows
  - Human-to-human and AI-powered chat integration
